Introduction

Javascript (JS) is a lightweight language used mainly for the logic area of web development, alongside its brother Typescript (TS). They are very similar with each other: while javascript is used for interactive web pages, typescript is strongly typed and can check errors when assigning types.

Here are some frameworks that use either JS or TS:

  • React
  • Angular
  • Meteor

Javascript

Do not confuse Javascript with Java: they are two completely different languages! While both of them are registered trademarks of Oracle in the US, their sintaxis, use and semantics are very different.

Read Javascript documentation. Reading a bit is always good!

Learn abour JS and Java differences. Its important you dont mistake one another.

How to print things

Although JS is called 'begginer friendly' by some people, things get harder the more you dive into it. To make it simple, we will print a single line:

console.log("Hello World!")

You can try coping and pasting it in a online JS compiler.

Variables

Variables are used to contain a value in an application.

You can start variables with letters, an underscore (_), a dollar sign ($), numbers (0-9), and because JS is case-sensitive, we can go from "A" to "Z" and from "a" to "z". Those are a lot of possible variables!

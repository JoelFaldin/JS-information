Introduction

Javascript (JS) is a lightweight language used mainly for the logic area of web development, alongside its brother Typescript (TS). They are very similar with each other: while JS is used for interactive web pages, TS is strongly typed and can check errors when assigning types.

Here are some frameworks that use either JS or TS:

  • React
  • Angular
  • Meteor

But, what are the differences between JS and TS? Here we have the most important ones:

  • TS is a 'superset' of JS, which means it understands all JS features and include new ones.
  • TS needs to be compiled to JS before it can run.
  • TS is strongly typed, this means you must exlpicitly declare the datatypes when creating a variable, and they can not be changed. On the other hand, JS is dynamically typed: you dont need to declare the datatype and it can be changed anywhere.

How to print things

Although JS is called 'begginer friendly' by some people, things get harder the more you dive into it. To make it simple, we will print a single line:

console.log("Hello World!")

You can try coping and pasting it in an online JS compiler.

Variables

Variables are used to contain a value in an application.

You can start variables with letters, an underscore (_), a dollar sign ($), numbers (0-9), and because JS is case-sensitive, we can go from "A" to "Z" and from "a" to "z". Those are a lot of possible variables!

There are 3 ways of declaring a variable:

The first one is by using the 'var' keyword. It declares the variably both globally and locally:

var x = 42;

The second way is by the 'let' keyword. It declares the variable as 'block scoped':

let x = 12;

And lastly, we can just asign a varibale:

x = 42;

Note that this last way of declaring a variable will give us a JS error and we should not use it.

Datatypes

A datatype gives us details about the variable we are using. It is universally important in all programming languages, and JS features 8 different datatypes:

  • String ("Hello world!")
  • Number (from 9 to 3.14159)
  • Bigint (a very large number)
  • Boolean (true or false)
  • Undefined (literally undefined)
  • Null (different from null or NULL)
  • Symbol (guaranteed to be unique)
  • Object (it can be an objet, an array or a date)

Some characteristics of JS makes datatypes interact in different ways.

For example, when adding a number to a string, JS will work with the number as a string:

let x = 16 + "keybards"
console.log(x)

If we compile this, we will get the result 16car.
